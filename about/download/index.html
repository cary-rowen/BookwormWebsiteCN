以下提供 Bookworm 最新稳定版的下载链接，资源由 NVDA 中文站 提供。

安装版（Setup）适合常规安装使用；便携版（Portable）解压后即可运行，也可以从移动存储设备启动。x86 版可在 32 位或 64 位 Windows 上运行，x64 版仅适用于 64 位 Windows。Bookworm 使用 Python 3.11 构建，支持 Windows 8.1 及更高版本；Windows 7 及更早版本不受支持。

更新内容

本地化更新

  • 西班牙语：添加了西班牙语翻译（@ericdq，#283）
  • 芬兰语：更新了翻译、文档及用户指南（@jkinnunen，#284、#316、#342）
  • 简体中文：更新了文档及界面翻译（@cary-rowen，#293、#369）
  • 葡萄牙语：翻译更新（@George-br，#296）
  • 法语：翻译更新（@sofquipeut，#313）
  • 格鲁吉亚语：翻译更新（@DraganRatkovich，#368）

功能与增强

  • 添加了百度在线 OCR 引擎（@cary-rowen，#334）
  • 通过与 NVDACN 合作，添加了 Vivo OCR 引擎（@cary-rowen，#338）
  • 增加了 TTS 媒体键控制，支持前台和全局两种模式（@cary-rowen，#328）
  • 在用户界面中添加了文本自动换行选项（@cary-rowen，#299）
  • 增加了按位置对书签进行排序的功能（@pauliyobo，#324）
  • 允许从 QRD 文件导入书籍元数据（@pauliyobo，#310）
  • 增加了“更新日志”菜单项（@cary-rowen，#337）

文档处理

  • 每当 EPUB 或 Word 文件被修改时，自动更新文档缓存（@pauliyobo，#289）
  • 修复了导致 .docx 文档无法打开的错误（@pauliyobo，#312、#315）
  • 保留原始 URI，以确保转换后的文档能正确恢复上次阅读位置（@cary-rowen，#340）
  • EPUB 专项修复：
    • 处理了 EPUB HTML 片段中缺失 <head> 或 <body> 标签的情况（@cary-rowen，#319）
    • 改进了 EPUB 表格的无障碍支持和语义结构（@cary-rowen，#362）
    • 针对格式错误的 HTML 内容在 _get_title_for_section 中增加了异常处理（@cary-rowen，#356）
    • 修正了 get_epub_html_item_by_href 中的拼写错误（@cary-rowen，#357）

导航与注解

  • 允许将注释添加到一行中的特定文本范围，而不是整行（@pauliyobo，#280）
  • 修复了注释跳转时的遗漏和循环问题，并改进了整行注释的定位行为（@cary-rowen，#336）
  • 修复了更改存储路径后导致文档丢失关联注释的问题（@pauliyobo、@cary-rowen，#349）
  • 解决了链接文本显示不全及脚注导航问题，并调整了光标位置（@cary-rowen，#318、#320、#322）

通用修复与改进

  • 改进了更新检查的版本比较逻辑（@cary-rowen，#301）
  • 提高了语音引擎加载的稳定性，并修复了配置不匹配问题（@cary-rowen，#339）
  • 为 http_resource 请求添加了 User-Agent（@pauliyobo，#347）
  • 使用独立的 TextFixerConfig 实例控制 ftfy 的文本修正过程（@cary-rowen，#298）
  • 修正了阅读器工具栏中放大和缩小按钮的标签（@cary-rowen，#292）

依赖与工具更新

  • 构建与环境：
    • 迁移至 uv 和 Ruff，并改进了构建系统及环境元数据（@cary-rowen，#351、#352）
    • 添加了 pytest CI 工作流（@cary-rowen，#355）
    • 添加了 WinGet 自动发布工作流（@cary-rowen，#359）
    • 配置了 Dependabot 以实现自动更新（@cary-rowen，#360）
  • 依赖库升级：
    • 升级了多项依赖以解决弃用警告（@cary-rowen，#358）
    • 将 urllib3 升级至 2.6.3（@dependabot、@cary-rowen，#345、#350、#353、#354）
    • 将 jinja2 升级至 3.1.6（@dependabot，#291、#308）
    • 将 requests 从 2.32.3 升级至 2.32.4（@dependabot，#325）
    • 将 mammoth 从 1.7.1 升级至 1.11.0（@dependabot，#344）
    • 升级了 GitHub Actions 依赖组中的多项依赖（@dependabot，#361、#365）
