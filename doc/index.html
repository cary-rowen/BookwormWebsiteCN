Bookworm 用户指南

介绍

Bookworm 是一款文档阅读器，可让您通过灵活、简洁且具有良好无障碍支持的界面阅读 PDF、EPUB、MOBI 以及许多其他格式的文档。

Bookworm 提供了丰富的阅读工具。您可以搜索文档，为感兴趣的内容添加书签、高亮和注释，使用文本转语音朗读，使用 Bookworm 书架管理文档，打开网页文章，还可以通过光学字符识别（OCR）将扫描版文档转换为纯文本。

Bookworm 运行在 Microsoft Windows 上，可以很好地配合 NVDA、JAWS 等屏幕阅读器使用。即使未启用屏幕阅读器，Bookworm 也能使用内置的文本转语音功能自行朗读内容。

功能特性

  • 支持超过 20 种文档格式，包括 EPUB、PDF、MOBI、Microsoft Word 文档、HTML、纯文本和 Markdown。
  • 支持结构化导航，可使用单键命令在标题、链接、列表、表格、引用和图片之间跳转。
  • 支持全文搜索，并可自定义搜索选项，包括使用正则表达式以及按页码或章节限定搜索范围。
  • 提供功能丰富且易于使用的注解工具。您可以添加命名书签、注释和高亮，在各类注解之间快速跳转，并将注释或高亮导出为纯文本、HTML 或 Markdown 文档。
  • 对于支持图形渲染的格式，Bookworm 提供两种页面查看方式：纯文本视图，以及经过完整渲染且可缩放的图像视图。
  • 支持光学字符识别（OCR），可从扫描版文档和图像中提取文本。Bookworm 集成了 Windows 内置 OCR、开源的 Tesseract 引擎、VIVO 通用 OCR 以及百度 AI 云 OCR 服务。
  • 内置网页正文提取器，可打开 URL 并自动提取页面正文。
  • 支持维基百科快速搜索，并可在 Bookworm 中打开维基百科文章。
  • Bookworm 书架可用于整理本地文档、导入文件或文件夹、搜索标题和已索引内容，并将文档打包供离线使用。
  • 对提供目录的文档格式提供完善的目录导航支持。
  • 支持使用文本转语音朗读书籍，并可通过语音配置文件自定义语音选项。
  • 支持使用标准的放大、缩小和重置命令缩放文本。
  • 支持将任何文档格式导出为纯文本文件。

安装

要在您的计算机上安装并运行 Bookworm，请访问 Bookworm 发布页面，下载最新版本。

当前版本需要 Windows 8.1 或更高版本。Windows 7 及更早版本不受支持。

Bookworm 提供三种版本：

  • 适用于 32 位或 64 位 Windows 的 32 位安装版
  • 适用于 64 位 Windows 的 64 位安装版
  • 可从 U 盘等移动存储设备直接运行的便携版

如果您的系统中安装了旧版 SAPI5 语音，并希望在 Bookworm 中使用这些语音，建议安装 32 位版本的 Bookworm，或使用 32 位便携版。

选择合适的版本下载。如果下载的是 Bookworm 安装版，请运行 .exe 文件并按照屏幕提示操作。如果选择便携版，请将压缩包解压到任意位置，再运行其中的 Bookworm 可执行文件。

用法

打开文档

您可以从“文件”菜单中选择“打开...”，也可以按 Ctrl+O。系统会显示“打开文件”对话框；找到要阅读的文档并选择“打开”，即可将其加载到 Bookworm 中。

在“文件”菜单中，您还可以按 Ctrl+N 打开新的 Bookworm 窗口，按 Ctrl+W 关闭当前文档，或按 Ctrl+P 固定当前文档。固定文档和最近打开的文档分别位于对应的子菜单中，也可以从这些子菜单中清空列表。

“文件”菜单还包含“导入”子菜单。“导入 QRD 文件”可导入 QRead QRD 文件中的阅读位置信息，并在保存的位置打开原始文档。“首选项...”用于打开 Bookworm 的首选项，也可以按 Ctrl+Shift+P 打开。

阅读器窗口

Bookworm 主窗口包含以下部分：

  1. “目录”：显示文档章节，便于浏览文档结构。使用导航键在章节之间移动，按 Enter 可跳转到指定章节。在文本视图中按 Ctrl+T 可将焦点移到目录。

  2. “文本视图”区域：显示当前页的文本。您可以使用常规文本导航命令阅读，也可以使用以下键盘快捷键在文档中导航：

  • Enter 或 Space：跳转到当前章节的下一页
  • Backspace：跳转到当前章节的上一页
  • Page Down 和 Page Up：在当前页内向前或向后移动较大一段文本
  • 当光标位于第一行时，连续按两次上箭头可跳转到上一页。
  • 当光标位于最后一行时，连续按两次下箭头可跳转到下一页。
  • Alt+Home：跳转到当前章节的第一页
  • Alt+End：跳转到当前章节的最后一页
  • Alt+Page Down：跳转到下一章节
  • Alt+Page Up：跳转到上一章节
  • F2：跳转到下一个书签
  • Shift+F2：跳转到上一个书签
  • F8：跳转到下一条注释
  • Shift+F8：跳转到上一条注释
  • F9：跳转到下一处高亮
  • Shift+F9：跳转到上一处高亮
  • Ctrl+Enter：执行当前位置的特殊动作。Bookworm 可跟随内部链接、在默认浏览器中打开外部链接、在可浏览对话框中显示表格，或打开嵌入图片。
  • Ctrl+Shift+Enter：跟随内部链接后返回上一个位置。
  1. 阅读进度滑块：在首选项中启用“显示阅读进度百分比”后，Bookworm 会在状态栏显示阅读进度，并允许您使用滑块按阅读百分比在文档中跳转。

结构化导航

当文档包含语义结构时，您可以直接在文本视图中按元素导航。按相应字母可移到下一个元素；按住 Shift 再按同一字母可移到上一个元素。

  • H：标题
  • 1 到 6：1 到 6 级标题
  • K：链接
  • L：列表
  • T：表格
  • Q：引用
  • I：图片或图形

您也可以从“文档”菜单中打开“元素列表...”，或按 Ctrl+F7。元素列表可显示标题、链接、列表、表格、引用和图片。激活某一项即可跳转到对应元素。

文档命令

“文档”菜单包含与当前文档相关的命令：

  • “文档信息...”在相关信息可用时显示元数据和文档统计信息。
  • “元素列表...”可按 Ctrl+F7 打开结构化元素列表。
  • “更改阅读模式...”可按 Ctrl+Shift+M 打开阅读模式对话框。可用模式取决于文档格式，可能包括默认、阅读顺序、物理布局、分页、逐章、纯文本和全文模式。
  • “渲染页面...”可在文档格式支持图形渲染时按 Ctrl+R 打开当前页的完整渲染图像。

书签、注释和高亮

Bookworm 允许您为打开的文档添加注解。您可以添加书签以记住文档中的特定位置，并在日后快速跳转到该位置。您还可以添加注释来记录想法或摘要，并高亮选中的文本以便稍后查看。

添加书签

阅读文档时，您可以按 Ctrl+B，或从“注解”菜单中选择“添加书签”，在当前光标位置添加书签。您也可以按 Ctrl+Shift+B，或从“注解”菜单中选择“添加命名书签...”，来添加命名书签。

查看书签

进入“注解”菜单并选择“已保存的书签...”，即可打开书签列表。激活列表中的任意项会立即跳转到该书签所在位置。要在文本视图中快速在书签之间跳转，请使用 F2 和 Shift+F2。

在已保存书签对话框中，按 F2 可重命名选中的书签，按 Delete 可将其删除。

添加注释

阅读文档时，您可以按 Ctrl+M，或从“注解”菜单中选择“添加注释...”。Bookworm 会提示您输入注释内容。输入内容并选择“确定”后，注释会添加到当前位置。如果已选中文本，注释会附加到选中的范围。

如果希望 Bookworm 在创建注释后询问标签，请在添加注释时按住 Shift。

进入含有注释的页面时，Bookworm 可以播放一个简短音效作为提示。此行为可在首选项的“注解”页面中更改。

添加高亮

选中文本后按 Ctrl+H，或从“注解”菜单中选择“高亮选中内容”，即可将所选文本保存为高亮。如果同一选区已被高亮，则会移除该高亮。如果选区与已有高亮重叠，Bookworm 可以扩展已有高亮。

如果希望 Bookworm 在创建高亮后询问标签，请在添加高亮时按住 Shift。使用 F9 和 Shift+F9 可移动到下一处和上一处高亮。

管理注释和高亮

从“注解”菜单中选择“已保存的注释...”或“已保存的高亮...”，即可打开已保存项目列表。激活列表中的任意项会立即跳转到对应位置。选择“查看”按钮可打开对话框，显示所选项目的标签和内容。

您可以按标签、章节或内容过滤已保存的注释或高亮。相关信息可用时，还可以按日期、页码、位置或书籍排序。按 F6 可移到过滤控件，按 F2 可编辑标签，按 Delete 可删除所选项目，按 Ctrl+C 可复制注解文本。

导出注释和高亮

Bookworm 允许您将已保存的注释和高亮导出为纯文本文件、HTML 文档或 Markdown 文档。

要导出注释或高亮，请按照以下步骤操作：

  1. 在“注解”菜单中，进入“已保存的注释...”或“已保存的高亮...”。
  2. 找到“导出”按钮并按 Enter，或使用 Alt+X 快捷键打开导出菜单。

您可以根据需要选中或取消选中以下选项：

  • 包含书名：在最终输出文件中包含书名。
  • 包含章节标题：包含注解所在章节的标题。
  • 包含页码：包含注解所在页面的页码。
  • 包含标签：在输出中包含注解标签。

根据需要指定选项后，选择纯文本、HTML 或 Markdown 输出格式，再选择文件的保存位置和名称。选中“导出后打开文件”后，Bookworm 会在保存完成后自动打开输出文件。

Bookworm 书架

Bookworm 书架是用于整理阅读文档的本地书库。您可以从“文件”菜单的“书架”子菜单中选择“打开书架”。您也可以选择“添加到本地书架...”，将当前打开的本地文档添加到书架。

添加或导入文档时，Bookworm 会让您选择阅读清单和收藏，并询问是否将文档加入全文搜索索引。首选项的“书架”页面中提供了自动将打开的书籍添加到本地书架的选项。

本地书架包含“最近添加”“正在阅读”“想要阅读”“最爱”“阅读清单”“收藏”和“作者”等分类。书架窗口的“文件”菜单允许您按 Ctrl+O 导入文档、从文件夹导入文档、搜索书架、存档文档，以及清除无效文档。

在书架文档列表中：

  • 按 Enter 可在新的 Bookworm 窗口中打开选中的文档。
  • 当可重命名时，按 F2 可重命名选中的文档。
  • 按 F5 可刷新列表。
  • 按 Ctrl+F 可将焦点移到快速过滤框。
  • 焦点位于文档列表时直接输入文本，可按标题过滤。
  • 按 Alt+Left 可从嵌套文件夹或容器返回上一级。

书架上下文菜单可在 Bookworm 中打开文档、使用系统查看器打开文档、编辑标题、显示文档信息、选择“编辑阅读列表/收藏...”、切换“正在阅读”“想要阅读”或“最爱”状态，以及从书架中移除文档。

“搜索书架...”命令会搜索文档标题和已索引内容。您可以从搜索结果直接打开匹配的文档，并跳转到匹配内容所在的页面和位置。

打开 URL 和使用维基百科

Bookworm 可以将网页作为可读文档打开。从“Web 服务”菜单中选择“打开 URL”并输入 URL，或按 Ctrl+Shift+U 打开剪贴板中的 URL。Bookworm 会加载页面，并尽可能提取网页正文。网页支持纯文本和全文等阅读模式。

“维基百科快速搜索”命令位于“Web 服务”菜单中，也可以按 Ctrl+Shift+W 调用。如果已选中文本，Bookworm 会将其作为搜索词。您可以选择维基百科语言、查看摘要，并在 Bookworm 或浏览器中打开文章。选中文本时，上下文菜单中还会出现“在维基百科上查询”。

光学字符识别（OCR）

Bookworm 可以使用 OCR 功能从图像和扫描版文档中提取文本。这尤其适合将图像型 PDF 或文档照片转换为可阅读、可搜索的文本。Bookworm 支持多个 OCR 引擎，您可以根据需要选择最合适的引擎。

您可以从菜单栏的“OCR”菜单访问 OCR 功能。主要功能包括：

  • “扫描当前页...”（F4）：对文档的当前页执行 OCR。
  • “自动 OCR”（Ctrl+F4）：在您切换到新页面时自动执行 OCR。
  • “更改 OCR 选项...”：更改当前书籍的 OCR 选项。
  • “扫描到文本文档...”：扫描页面并将识别出的文本保存到 .txt 文件。
  • “图片转文字...”：选择计算机中的图像文件，并将识别出的文本作为虚拟文档在 Bookworm 中打开。

OCR 选项对话框允许您选择主要识别语言；当所选引擎支持多语言识别时，还可以选择辅助识别语言。您也可以更改提供给 OCR 引擎的图像分辨率、启用图像增强，并使所选设置在关闭当前书籍前持续生效。

可用的图像预处理过滤器包括提高图像分辨率、二值化、将包含两页的扫描图拆分为单页、合并图像、模糊、纠偏、腐蚀、膨胀、锐化和反色。可用过滤器和引擎特定选项取决于所选 OCR 引擎。

Bookworm 支持以下 OCR 引擎：

Windows 10/11 OCR

如果您使用 Windows 10 或更高版本，Bookworm 可以使用操作系统内置的 OCR 引擎。该引擎在可用时是默认引擎，无需额外设置。对于系统中已安装的语言，它通常能提供良好的识别结果。

Tesseract OCR 引擎

对于需要更广泛语言支持的用户，Bookworm 支持集成 Tesseract OCR 引擎。Tesseract 是由 Google 维护的强大开源 OCR 引擎。

如果尚未为 Bookworm 安装 Tesseract，您可以在应用内下载并设置：

  1. 进入“文件 > 首选项...”，选择“OCR”页面。
  2. 在“Tesseract OCR 引擎”区域中，点击“下载 Tesseract OCR 引擎”按钮并按照提示操作。
  3. 安装完成后，您可以点击“管理 Tesseract OCR 语言”按钮来管理语言。

VIVO 通用 OCR 引擎（通过 NVDA-CN）

通过与 VIVO（vivo.com.cn）和 NVDA 中文社区（NVDACN）的合作，Bookworm 提供免费的 VIVO OCR 引擎，可高质量识别中文和英文内容。

要使用 VIVO OCR 引擎，您需要一个免费的 NVDA-CN 账户。

设置 VIVO OCR

  1. 前往 NVDA-CN 注册页面创建账户：https://nvdacn.com/admin/register.php。
  2. 点击验证邮件中的链接完成邮箱验证。
  3. 通过“文件 > 首选项...”或 Ctrl+Shift+P 打开 Bookworm 首选项。
  4. 转到“OCR”设置页面，在“VIVO OCR 引擎”区域输入用户名和密码。
  5. 在“默认 OCR 引擎”列表中选择“VIVO OCR”作为默认 OCR 引擎。

配置完成后，Bookworm 会使用 VIVO 引擎执行 OCR。账户相关问题可联系 NVDA-CN 团队：support@nvdacn.com。

百度 AI 云 OCR

Bookworm 集成了百度 AI 云 OCR 服务，在处理中英文混排或复杂版式时可提供较高的识别准确率。该在线服务提供标准版和高精度版两种引擎。

要使用百度 OCR 引擎，您需要获取免费的 API Key 和 Secret Key。

设置百度 OCR

  1. 在百度 AI 云 OCR 页面注册账户并获取密钥。
  2. 通过“文件 > 首选项...”或 Ctrl+Shift+P 打开 Bookworm 首选项。
  3. 转到“OCR”设置页面，在“百度 OCR 引擎”区域输入 API Key 和 Secret Key。
  4. 在“默认 OCR 引擎”列表中选择“百度通用 OCR（标准版）”或“百度通用 OCR（高精度版）”。

配置完成后，Bookworm 会使用百度引擎执行 OCR。

朗读

Bookworm 可以使用系统中已安装的 TTS 语音朗读当前文档。按 F5 开始朗读，按 F6 暂停或恢复朗读，按 F7 完全停止朗读。

您可以通过两种方式配置语音：

  1. 使用语音配置文件：语音配置文件包含自定义的朗读设置，您可以随时启用或停用。可从“朗读”菜单打开语音配置文件，也可以按 Ctrl+Shift+V。Bookworm 内置了一些示例语音配置文件。
  2. 全局语音设置：未启用语音配置文件时，Bookworm 默认使用这些设置。您可以在应用首选项中配置全局语音设置。

朗读过程中，按 Alt+Left Arrow 可后退到上一段，按 Alt+Right Arrow 可快进到下一段。

媒体键行为

媒体键可用于执行主要的文本转语音操作：

  • 播放/暂停：切换 TTS 的播放、暂停和恢复。
  • 下一曲：快进到下一段。等同于 Alt+Right Arrow。
  • 上一曲：后退到上一段。等同于 Alt+Left Arrow。

默认情况下，只有当 Bookworm 窗口拥有焦点时，媒体键才由 Bookworm 处理。您可以在首选项的“阅读”页面中选中“启用全局媒体键”来开启全局媒体键。这样 Bookworm 在后台运行时也可以控制播放。

同时运行多个媒体应用时，媒体键功能可能不稳定。

配置阅读风格

除了语音设置之外，Bookworm 还允许您微调阅读行为。以下所有设置都可以在应用首选项的“阅读”页面中找到。

  • 开始朗读时：此设置决定 Bookworm 开始朗读当前文档时执行的操作。您可以选择“朗读整本书”“朗读当前章节”或“朗读当前页”。
  • 开始朗读位置：此选项决定从何处开始朗读。可以从光标位置或当前页开头开始。
  • 朗读过程中：这一组选项控制 Bookworm 在朗读过程中的行为。
  • 读出页码：导航到每一页时，文本转语音会读出页码。
  • 提示章节结束：章节结束时，文本转语音会进行提示。
  • 请求切换到适合当前书籍语言的语音：当所选 TTS 语音与当前文档的语言不同时，此选项决定 Bookworm 是否提示您切换语音。
  • 高亮显示朗读内容：启用后，当前朗读的文本会在界面中高亮显示。
  • 选中朗读内容：启用后，当前朗读文本会被选中。例如，您可以在朗读过程中按 Ctrl+C 复制当前朗读的段落。

“阅读”页面还包含在文本中导航时提供图片反馈的选项，包括导航到图片时播放声音，以及在图片导航中包含替代文本为空的图片。

连续阅读模式

除了 Bookworm 内置的文本转语音功能外，您还可以使用屏幕阅读器的连续阅读功能，也称为“全文朗读”。Bookworm 通过“连续阅读模式”支持此功能。该模式默认启用，您可以在应用首选项的“阅读”页面中禁用。启用后，屏幕阅读器读到页尾时，Bookworm 会自动翻页。

请注意以下限制：

  • 如果到达空白页，连续阅读会中断。请移到非空白页，再重新启动屏幕阅读器的连续阅读功能。
  • 当光标移到当前页最后一个字符时，会立即切换到下一页。

查看当前页的完整渲染版本

当文档支持图形渲染时，Bookworm 允许您查看当前页的完整渲染版本。打开文档后，可以按 Ctrl+R，或从“文档”菜单中选择“渲染页面...”。该视图称为“渲染视图”，以区别于默认的文本视图。

在渲染视图中，您可以使用常规缩放命令：

  • Ctrl+=：放大
  • Ctrl+-：缩小
  • Ctrl+0：重置缩放级别

您也可以使用前文提到的文档导航命令在渲染视图中翻页。按 Escape 可关闭此视图并返回文本视图。

当 Ctrl+Enter 打开嵌入图片时，图片对话框提供以下命令：

  • Ctrl+=：放大
  • Ctrl+-：缩小
  • Ctrl+0：实际大小
  • Ctrl+S：保存图片
  • Ctrl+C：复制图片
  • Escape、Ctrl+W 或 Alt+C：关闭图片对话框

当 Ctrl+Enter 打开表格时，Bookworm 会在可浏览的 HTML 对话框中显示该表格。

跳转到指定页

要跳转到当前文档中的指定页面，请按 Ctrl+G，或从“搜索”菜单中选择“页码跳转”，打开“跳转页码”对话框。输入页码后，Bookworm 会跳转到对应页面。此对话框还会显示当前文档的总页数。

对于提供页标签的文档，例如某些 PDF 文档，您可以按 Ctrl+Shift+G，或从“搜索”菜单中选择“标签跳转...”，然后在“按标签跳转”对话框中输入页标签。

要跳转到当前页或文档中的某一行，请按 Ctrl+L，或从“搜索”菜单中选择“行跳转”。

搜索文档

要在当前打开的文档中查找特定术语或一段文本，请按 Ctrl+F 打开“搜索文档”对话框。此对话框允许您输入要搜索的文本，并配置搜索过程。可用选项包括：

  • 区分大小写：搜索时会考虑搜索词中字母的大小写。
  • 整词匹配：搜索词必须作为完整单词出现，而不是作为另一个单词的一部分。
  • 正则表达式：将搜索词作为正则表达式处理。启用此选项时，整词匹配会被禁用。
  • 搜索范围：根据文档情况，您可以将搜索限制在某些页面、特定章节或文本范围内。

在“搜索文档”对话框中选择“确定”后，会显示搜索结果对话框。激活列表中的任意项会立即跳转到对应位置，并高亮显示搜索词。

如果您已经关闭搜索结果窗口，可以按 F3 和 Shift+F3 分别移动到上一次搜索结果中的下一个和上一个匹配项。

管理文件关联

应用首选项“常规”页面中的“管理文件关联”按钮，可帮助您管理哪些文件类型与 Bookworm 关联。关联后，在 Windows 资源管理器中打开相应文件时，Windows 会默认使用 Bookworm。此对话框会在首次启动时显示，并且仅在安装版中可用；便携版不提供文件关联管理。

打开文件关联管理器后，您会看到以下选项：

  • 关联所有格式：使 Windows 默认使用 Bookworm 打开所有受支持的文件类型。
  • 取消所有支持文件类型的关联：移除之前注册的文件关联。
  • 每种受支持文件类型对应一个按钮：选择按钮即可关联相应文件类型。

更新 Bookworm

默认情况下，Bookworm 会在启动时检查新版本，以便您及时获取更新。您可以在应用首选项中关闭自动检查，也可以从“帮助”菜单中选择“检测更新”手动检查。

无论采用哪种方式，当发现新版本时，Bookworm 都会询问您是否要安装。如果选择“是”，应用会下载更新包，并显示下载进度对话框。下载完成后，Bookworm 会提示将重启应用以完成更新。选择“确定”即可完成更新过程。
